

# 3. Key Features (주요 기능)

## 🏠 메인 페이지
- 주요 컨텐츠 카드 뉴스 제공(인기 · 추천 · 신규 셀러)
- 카테고리별 탐색 기능
- 검색 기능 제공(검색 시 리스트 페이지 이동)

---

## 🔍 리스트 / 검색
- 검색 결과 리스트 제공
- 필터링 · 정렬 · 페이징 기능

---

## 👤 사용자 기능(User)
- 회원가입 · 로그인 · 아이디/비밀번호 찾기
- 마이페이지(회원 정보 관리 · 리뷰 내역 · 즐겨찾기 내역 · 회원 탈퇴)
- 즐겨찾기 기능
- 리뷰 작성

---

## 🛒 셀러 기능(Seller)
- 셀러 가입
- 제품 관리
- 정산 관리
- 리뷰 관리(리뷰 답글 작성)

---

## 🛠️ 관리자 기능(Admin)
- 사용자 관리
- 셀러 관리
- 컨텐츠 관리
- 리뷰 관리
- 결제(환불) 관리
- 공지사항 관리
